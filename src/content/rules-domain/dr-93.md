All input ID fields (propertyId, cursor, propertyTypeId, etc.) MUST use a shared zUuid schema: z.string().length(36).regex(/^[0-9a-f]{8}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{4}-[0-9a-f]{12}$/i). This ensures consistent UUID format validation across all domains and prevents malformed IDs from reaching the database layer.
