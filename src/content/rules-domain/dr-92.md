properties.propertyTypeId is a nullable FK to property_types.id with explicit onDelete: "set null". If a property type is hard-deleted, all properties referencing that type survive with propertyTypeId = NULL (uncategorized). The field is already nullable, so this is safe — no data loss, no orphan crash. The Drizzle schema MUST declare onDelete: "set null" explicitly on the propertyTypeId reference.
